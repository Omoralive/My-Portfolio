Projects

Chess Game

I have built a chess game using Java OOP, creating my own Java chess engine from scratch. This game supports the full set of chess pieces, allows players to take turns, and even has a restart option when the game ends.

  • Implements all chess pieces (Pawn, Rook, Knight, Bishop, Queen, King)
  • Turn-based gameplay: players alternate between White and Black
  • Game Over: the game ends when a King is captured
  • Restart: option to restart the game once it ends
